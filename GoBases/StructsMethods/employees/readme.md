# Exercício 2 - Employee

Uma empresa precisa fazer um bom gerenciamento de seus funcionários. Para isso, criaremos um pequeno programa que nos ajudará a gerenciar corretamente esses funcionários. Os objetivos são:

1. Criar uma estrutura Person com os campos ID, Name, DateOfBirth.

2. Criar uma estrutura Employee com os campos: ID, Position e uma composição com a estrutura Person.

3. Criar um método para a estrutura Employee chamado PrintEmployee(), que imprimirá os campos de um funcionário.

4. Instanciar na função main() uma Person e um Employee carregando seus respectivos campos e, finalmente, executar o método PrintEmployee().
